# Heading

A heading at a specified level.

<Preview>
  <Story name="Demo">
    <Heading level={1} style={"beta"} centred={true} noMargin={false}>
      Simple heading
    </Heading>
  </Story>
</Preview>

## Properties

| Name             | Type                  | Default | Description|
|:-----------------|:----------------------|:--------|:-----------|
| `children`       | `Node`                | `null`  | The heading content.|
| `level`          | `Integer`             | 1       | The heading level ie h1, h2, h3 a level of 0 implements a span tag.|
| `style`          | `String`              | `null`  | Heading style taken from theme. If no style is specified, one will be selected depending on level: `alpha` for `0` and `1`, `beta` for `2`, `gamma` for `3`, `delta` for `4`, `epsilon` for `5` and `zeta` for `6`.|
| `noMargin`       | `Boolean`             | `false` | Select `true` to remove margins.|
| `centred`        | `Boolean`             | `false` | Select `true` to centre.|


## Theme

If you want to provide a theme via context, the component key is `ATHeading`.

| Name             | Description|
|:-----------------|:-----------|
| `centred`        | Centred heading                    |
| `noMargin`       | Heading with no margin             |
| `hero`           | Large hero heading for page header |
| `hero-leader`    | Leader for a hero haeding          |
| `hero-subtitle`  | Subtitle to a hero heading         |
| `hero-date`      | Date for a hero heading            |
| `alpha`          | Level 1 heading                    |
| `alpha-leader`   | Leader to a level 1 heading        |
| `alpha-subtitle` | Subtitle to a level 1 heading      |
| `alpha-date`     | Date for a level 1 heading         |
| `beta`           | Level 2 heading                    |
| `beta-leader`    | Leader to a level 2 heading        |
| `beta-subtitle`  | Subtitle to a level 2 heading      |
| `beta-date`      | Date for a level 2 heading         |
| `gamma`          | Level 3 heading                    |
| `delta`          | Level 4 heading                    |
| `epsilon`        | Level 5 heading                    |
| `zeta`           | Level 6 heading                    |
